### 1. 의존성 주입(DI, Dependency Injection)
- DI는 Dependency Injection의 약자
- 우리말로는 '의존성 주입'이라는 단어로 번역되어 사용
  
  
#### 1.1 의존(Dependency)
```java
public class FilePrinter {
	public void print(String filePath) throws IOException {
		// 의존하는 타입을 로컬 변수(br 변수)로 정의
	}
}
```
